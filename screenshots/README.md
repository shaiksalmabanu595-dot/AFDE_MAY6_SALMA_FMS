# Screenshots

Place screenshots of the running application here. Recommended captures:

- `dashboard.png` — Dashboard page showing total feedback, average rating, distribution chart, and recent feedback
- `submit-feedback.png` — Submit Feedback form with the rating stars
- `feedback-list.png` — All Feedback page with the grid of cards
- `feedback-detail.png` — Detail view of a single feedback entry
- `edit-feedback.png` — Edit form pre-filled with existing values
- `search-filter.png` — Search results with active filters applied
- `swagger-ui.png` — FastAPI auto-generated Swagger UI at `/docs`
- `postman-get.png` — Postman testing the `GET /feedback` endpoint
- `postman-post.png` — Postman testing the `POST /feedback` endpoint
- `postman-search.png` — Postman testing the `GET /search` endpoint

How to capture screenshots:

- **macOS:** Cmd + Shift + 4 (region) or Cmd + Shift + 3 (full screen)
- **Windows:** Win + Shift + S (Snipping Tool)
- **Linux (GNOME):** Shift + Print Screen
